Class: GoogleCompute

GoogleCompute

Interfaces with GCP for all compute related functionality.

new GoogleCompute(project, name, stage, imageName, zone, zoneLetter, ports, ports)

Parameters:
Name Type Description
project String The project name to associate with the instance.
name String The name of the compute instance.
stage String The stage of the project. Typically `production`, `staging` or `development`.
imageName String The name of the tagged Docker image.
zone String The name of the zone in which to allocate the instance.
zoneLetter String The zone letter for the instance.
ports String The ports to oen for this compute instance.
ports Array.<Number> | Number The port numbers to open. If no ports provided, then ALL ports are opened.
